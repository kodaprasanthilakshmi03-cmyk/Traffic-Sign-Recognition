# Traffic Sign Detection using CNN

A deep learning web app to classify traffic signs using an image uploaded by the user.

## 🔍 Features
- Upload image and get prediction
- Info about dataset and model
- Model accuracy and confusion matrix
- Built with Flask + Keras + HTML

## 🧠 Dataset
German Traffic Sign Recognition Benchmark (GTSRB) – 43 classes

## 🛠 Tech Stack
- Python
- Flask
- TensorFlow / Keras
- HTML / Bootstrap

## 🧪 Run Locally

```bash
pip install -r requirements.txt
python app.py
